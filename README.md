# 洛谷比赛监控提醒工具

一个Python后台程序，用于监控洛谷比赛列表，当比赛从"未开始"变为"进行中"时自动弹窗提醒。

## 功能特性

- 🔍 **定时检查**：在每个整点和半点自动检查比赛状态（延迟5秒避免网络误差）
- 🔔 **弹窗提醒**：比赛开始时弹出美观的提醒窗口，支持一键打开比赛页面
- 💾 **状态持久化**：本地保存比赛状态，准确检测状态变化
- 🎨 **美观界面**：自定义弹窗设计，蓝色主题，中文优化

## 安装依赖

```bash
pip install -r requirements.txt
```

## 使用方法

### 前台运行（有控制台窗口，方便调试）

```bash
python main.py
```

### 后台运行（无控制台窗口）

```bash
pythonw main.py
```

### 停止程序

- 前台运行：按 `Ctrl+C`
- 后台运行：在任务管理器中找到 Python 进程并结束

## 工作原理

1. 程序启动时立即执行一次检查
2. 计算下一个整点或半点（如当前是 14:23，则下次是 14:30:05）
3. 等待到目标时间 + 5 秒后执行检查
4. 获取洛谷比赛列表，解析比赛状态
5. 与本地保存的状态进行比对
6. 当检测到某比赛从"未开始"变为"进行中"时，弹出窗口提醒
7. 更新本地数据，继续循环等待

## 弹窗功能

- **打开比赛页面**：点击后直接打开浏览器访问比赛页面
- **知道了**：关闭弹窗
