## 项目概述
创建一个Python后台程序，监控洛谷比赛列表，当检测到比赛从"未开始"变为"进行中"时弹窗提醒。

## 技术栈
- **requests** - 网页数据获取
- **tkinter** - Windows弹窗提醒
- **json** - 本地数据持久化

## 项目结构
```
luogu-contest-alert/
├── main.py          # 主程序入口（整点半点检查）
├── crawler.py       # 网页爬取模块
├── storage.py       # 数据存储模块
├── notifier.py      # 弹窗提醒模块
├── config.py        # 配置文件
├── requirements.txt # 依赖列表
├── .gitignore       # Git忽略文件
├── README.md        # 项目说明
└── data/
    └── contests.json # 本地存储的比赛数据
```

## 核心功能
1. **定时检查**：每个整点和半点自动检查（延迟5秒）
2. **状态检测**：比对本地数据，检测"未开始"→"进行中"的变化
3. **弹窗提醒**：美观的提醒窗口，支持一键打开比赛页面
4. **后台运行**：支持pythonw后台运行，无控制台窗口

## 运行方式
```bash
# 安装依赖
pip install -r requirements.txt

# 前台运行
python main.py

# 后台运行（无控制台窗口）
pythonw main.py
```

## 实现要点
- 从 `window._feInjection` 解析JSON数据获取比赛信息
- 根据当前时间与比赛开始/结束时间计算状态
- 使用线程显示tkinter弹窗，避免阻塞主程序
- 本地存储比赛状态，确保重启后仍能正确检测变化
